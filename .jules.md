# Jules Instructions for 3D_Visual_Effects_Explorer

## Core Mission
Maintain and expand the 3D Visual Effects Explorer. Focus on high-fidelity visual aesthetics, spatial intelligence, and performant 3D rendering.

## Key Priorities
- **Gaussian Splatting:** When improving the renderer, prioritize optimizations for `.splat` and `.ksplat` files.
- **Mesh Painting:** Enhance the `EnhancedDetailShader.ts` with more complex procedural textures and vertex animations.
- **Aesthetic First:** If a performance fix compromises the visual "beauty" of an effect, prioritize finding a way to keep both.
- **Testing:** Ensure all new shaders have corresponding `.test.ts` files using the existing Three.js mocking patterns.

## Avoid
- Repetitive minor maintenance sessions. If you find multiple small issues, group them into a single "Code Health" session.
- Removing comments from shader code unless they are objectively incorrect.
